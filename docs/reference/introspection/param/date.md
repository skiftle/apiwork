# Date

[GitHub](https://github.com/skiftle/apiwork/blob/main/lib/apiwork/introspection/param/date.rb#L22)

Date param representing date values (year, month, day).

**Example: Basic usage**

```ruby
param.type         # => :date
param.scalar?      # => true
param.date?        # => true
```

**Example: Capabilities**

```ruby
param.formattable? # => false
```

**Example: Enum**

```ruby
if param.enum?
  param.enum      # => ["2024-01-01", "2024-12-31"]
  param.enum_reference? # => false
end
```
